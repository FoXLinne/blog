随便说几句

临近毕业，虽然还是很忙，但怎么说也没去年那么忙了，于是我又把博客重新开了起来。

这次依然选择用 GitHub Pages 托管静态网页，主要是更新了一下域名——这个新域名，大概未来很多年都不会再变了吧。

由于用的是 Hexo，需要从源文件生成静态页面，之前换过几次电脑，那些源文件基本都找不到了（本来也没多少内容哈哈），不过网页本身还保留着，所以我就把之前的文章简单复制搬运了过来。

这次翻新刚好也升级了 Hexo 和主题的版本，很多参数和插件也得重新配置，慢慢来吧。

记得上一次认真搭建博客已经是两年前的事了。那时候带着对新事物的好奇与期待，连夜埋头苦干，把一间毛坯房一点点装点得还算精致，为了不让网站显得太空，还装模作样写了几篇文章。虽然一两个月后，我就渐渐忘了维护这回事，但对于尚不懂编程的我来说，研究这种看似很有技术含量的东西真的很快乐。

两年前，AI 的发展远没有现在这么迅猛，那时它基本只能用来回答简单问题、聊聊天，或者帮忙写水课的论文，而如今，借助 AI 的讲解，普通人完全可以绕开那些复杂难懂的教程和文献，只要稍有点耐心，就能通过一步步的询问，搭建起属于自己的博客。与此同时，Hexo 虽然已经算是比较简单的工具了，但依然存在着手动部署这类有点儿烦人的操作，现在只需借助 OpenClaw 之类的工具，完全可以用一句简单的自然语言指令让它自动就完成部署。不得不让人感慨，时代真的变了。

眼下我正在思考，该把博客源文件托管在哪里，才能在多平台编辑都能很便捷的同时还兼顾远程部署，等到这套流程能顺下来，往博客里添加内容应该会更频繁一些吧。

最近还入手了 Ulysses，专门用来写 Markdown 文章，偶尔也写写小说——毕竟花了钱，不用总觉得亏了。

折腾这一圈，更新域名、配置工具，无非是想把“写作”这件事时刻带在身边，让自己少一点偷懒的借口。

总之一切又回到了起点，还请各位多多指教。
